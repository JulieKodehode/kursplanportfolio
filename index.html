Velkommen til min side

Hei! Jeg heter Julie, og jeg er veileder på Webutvikler kurs via Jobloop. På kurset har vi lært språkene HTML, CSS og JavaScript, og jeg har brukt disse språkene og ulike teknikker i de ulike språkene for å lage denne nettsiden. På kurset har vi fått erfaring med verktøy som Stackblitz, GitHub, Vercel og Figma som hjelper oss å planlegge, skrive og publisere prosjekter vi har laget

Mine større prosjekter

Parallax

I dette prosjektet har jeg tatt i bruk parallax-teknikker for å få en effekt av dybde når brukeren scroller på siden. Dette er oppnådd med HTML som strukturerer seksjonene, CSS som setter faste bakgrunnsbilder og definerer lagdeling, og litt JavaScript for å justere scroll-effekten slik at tekst og bilder beveger seg i ulik hastighet.

Gallery

I dette prosjektet har jeg tatt i bruk CSS Grid for å få en responsiv og ryddig bildevisning på tvers av skjermstørrelser. HTML brukes til å plassere bildene i en strukturert grid-container, CSS definerer rader og kolonner som tilpasser seg fire forskjellige breakpoints, og JavaScript kan utvides for å legge til interaktivitet hvis ønskelig.

Sign-up form

I dette prosjektet har jeg tatt i bruk JavaScript-stegvis navigasjon for å få et skjema som guider brukeren gjennom flere steg. HTML definerer input-feltene for hvert steg, CSS brukes til å skjule og vise riktig del av skjemaet, og JavaScript styrer logikken ved å validere felter og vise neste steg først når de er korrekt fylt ut.

Animasjons prosjekter

I dette prosjektet har jeg tatt i bruk CSS-animasjoner kombinert med JavaScript for å få bevegelige elementer som kan styres av brukeren. HTML gir strukturen med en sirkel og kontrollknapper, CSS definerer selve animasjonen med @keyframes, og JavaScript brukes til å starte, pause og resette animasjonen ved å endre CSS-egenskaper dynamisk.

Målenhet Konverterer Prosjekt

I dette prosjektet har jeg tatt i bruk JavaScript-funksjoner for å få en dynamisk konvertering mellom ulike måleenheter. HTML brukes til input-felt og nedtrekksmenyer, CSS gir struktur og stil, mens JavaScript henter verdien, gjør beregningen og viser resultatet direkte på siden.

Gjøremålsliste

I dette prosjektet har jeg tatt i bruk DOM-manipulasjon for å få en interaktiv liste som oppdateres med nye oppgaver. HTML gir grunnstrukturen med input-felt og liste, CSS gir en tydelig visuell fremstilling, og JavaScript brukes til å legge til <li>-elementer, lytte til knapper og oppdatere innholdet dynamisk.

    API-kall

    I dette prosjektet har jeg tatt i bruk API-integrasjon med fetch() for å få inn data utenfra og vise det på siden. HTML har en plassholder for teksten, CSS sørger for en enkel visuell fremstilling, og JavaScript sender et kall til et eksternt API, henter en tilfeldig kattefakta og legger det inn i DOM-en når brukeren trykker på knappen.
